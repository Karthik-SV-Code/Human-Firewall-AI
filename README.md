# Human Firewall AI | Digital Safety Assistant

**Human Firewall AI** is a cutting-edge, on-device digital security command center designed to turn human vulnerability into impenetrable defense. Using intuitive visual tools, real-time feedback, and dynamic simulations, this platform trains users to identify social engineering, phishing, and malvertising with an expert eye.

## 🚀 Vision
In an era where attackers use AI to craft indistinguishable phishing vectors, traditional email filters are no longer enough. The last line of defense is the user. **Human Firewall AI** bridges the gap between complex cybersecurity concepts and everyday digital hygiene through a beautifully crafted, glassmorphism-based UI.

## ⚡ Features
- **Zero-Latency Local Analysis:** All heuristics, text NLP parsing, and URL decompositions occur client-side. No data leaves your machine.
- **Threat Lab Simulations:** Safely experience simulated attacks in a sandboxed environment:
  - AI Voice Deepfake Scams
  - Interactive SMS Phishing (Smishing)
  - Business Email Compromise (BEC)
  - Ransomware / Malicious Download Traps
  - "Flip the Script" Attack Compiler
- **Security Instinct Arena:** A Tinder-style minigame where users train their muscle memory to rapidly classify incoming communications as safe or phishing, accompanied by immediate, tactical feedback overlays.
- **Expert Tool suite:**
  - Fact Sentinel
  - Message X-Ray
  - Link Inspector
  - Code Guard
  - Trust Score Generator

## 🛠 Tech Stack
- **HTML5 & CSS3** (Custom Animations, Grid/Flex Layouts)
- **Vanilla JavaScript** (Zero dependencies, pure DOM manipulation)
- **Tailwind CSS** (Via CDN for typography and utility scaffolding)
- **Lucide Icons** (SVGs)

## 🏃 Setup & Deployment
This project is completely statically hosted. No complex build pipelines or Node servers required.
1. Clone this repository.
2. Open `index.html` in your favorite modern browser (Chrome, Edge, Firefox, Safari).
3. The platform is instantly ready to use!
