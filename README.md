# 🔐 Human Firewall AI – Digital Safety Assistant

## 📌 Project Overview
**Human Firewall AI** is an interactive digital security platform designed to transform users into the strongest line of defense against modern cyber threats. By combining simulation-based learning, real-time analysis, and intuitive visual tools, the system empowers users to detect phishing, social engineering, and malicious content effectively.

---

## Hosted Link : https://human-firewall-ai.vercel.app/


---

## ❗ Problem Statement
In the modern digital landscape, cyberattacks such as phishing, deepfake scams, and social engineering have become increasingly sophisticated. Traditional security systems like spam filters are no longer sufficient to detect these threats.

The primary challenge is that **human users remain the most vulnerable point in cybersecurity** due to lack of awareness, training, and real-time decision-making skills.

---

## 💡 Project Description

### 🔍 Solution
Human Firewall AI provides a **hands-on, simulation-driven learning environment** that enables users to identify and respond to cyber threats in real time.

### ⚙️ How It Works
- Simulates real-world cyberattacks in a safe, controlled environment
- Provides instant feedback and threat analysis
- Uses client-side processing for fast and private execution
- Includes interactive tools for analyzing messages, links, and content

### 🚀 Key Features
- **Threat Lab Simulations**
  - AI Voice Deepfake Scams  
  - SMS Phishing (Smishing)  
  - Business Email Compromise (BEC)  
  - Ransomware & Malicious Downloads  

- **Security Instinct Arena**
  - Gamified interface for rapid threat classification
  - Immediate feedback to improve user awareness

- **Expert Tool Suite**
  - Fact Sentinel  
  - Message X-Ray  
  - Link Inspector  
  - Code Guard  
  - Trust Score Generator  

### 🌟 Key Advantages
- Converts complex cybersecurity concepts into **simple, visual experiences**
- Enables **active learning through interaction**
- Ensures **privacy with fully local processing**
- Requires **no backend or server setup**

## 🤖 Google AI Usage

### 🧠 Tools / Models Used
- Client-side Natural Language Processing (NLP)
- Heuristic-based threat detection techniques

### 🔗 How AI is Integrated
Currently, the system uses **local, rule-based and NLP-driven analysis** to simulate intelligent threat detection without relying on external APIs. This ensures:
- Zero latency
- Complete data privacy
- Offline functionality

### 🔮 Future Scope with Google AI
The platform can be further enhanced by integrating Google AI services such as:
- **Gemini API** for advanced message understanding
- **Vertex AI** for real-time phishing detection
- Machine learning models for improved threat classification
